# Admin templates for AWI guest researchers

Templates used when hosting external guests in the Earth System Complexity group at AWI Potsdam. Replacement blocks are written as `<PLACEHOLDER>` — fill in before sending.

## Files

- [employment-verification-template.md](employment-verification-template.md) — verification of employment at the external institution. Required for **every** incoming guest.
- [cooperation-declaration-UCM.md](cooperation-declaration-UCM.md) — content-related cooperation declaration between AWI and UCM (Prof. Marisa Montoya). Required for non-PhD, non-postdoc guests (e.g. professors).
- [cooperation-declaration-IGEO.md](cooperation-declaration-IGEO.md) — content-related cooperation declaration between AWI and IGEO (Dr. Jorge Alvarez-Solas). Required for non-PhD, non-postdoc guests (e.g. professors).
- [invitation-letter-template.md](invitation-letter-template.md) — general invitation letter from the AWI host to the incoming guest.
